JAVA基本语法

发表于 2020-10-18 更新于 2020-10-22 分类于 Java

尚硅谷java学习，java基础语法阶段

第一天

1、java程序分为编写——编译——运行三个过程
编写：用java语言写的程序，即源文件
编译：使用”javac 源文件名.java“ 将源文件编译成字节码文件
运行：使用”java 类名“进行解析运行字节码文件
2、在java源文件中可以声明多个类目，但是只有一个public 属性的class，且这个class的名字应该和源文件名字一样。
3、程序的入口是main()方法，且格式固定。
4、编译过程：编译以后会生成一个或多个字节码文件，字节码文件的文件名和java源文件中的类名一致，所以有几个类就会生成几个字节码文件。但是要运行的话只有包含main()方法的字节码文件才能运行，所以如果那个要运行那个类就要加入main()方法。
JDK = JRE + Java的开发工具(javac.exe , java.exe , javadoc.exe….)
JRE = JVM + Java核心类库

第二天
java命名规范

包名：多单词组成时，所有字母小写 xxxyyyzzz
类名、接口名：多单词组成时，所有单词首字母大写（驼峰式）XxxYyyZzz
变量名、方法名：多单词组成时，第一个单词全部小写，第二个单词开始首字母大写（小驼峰式）xxxYyyZzz
常量名：所有字母大写 XXXYYYZZZ
java定义的数据类型
一、变量按数据类来分⭐⭐⭐变量定义了一定要赋值，不会自动赋值0的。如果不赋值可能会报错
基本数据类型：
整形：byte / short / int / long（long型常量后面加上l/L不加为 int型） ⭐整形常量默认为 int型
浮点型： float / doule （float型常量末尾需加上 f/F，所以java默认用double型，又大，精度又好）⭐浮点型常量默认double型
字符型： char 用’’ 框起来，在运算的时候是转化为对应码的。
布尔型： boolean

引用数据类型：
        类（class）   String 在这里面   String （用“”框起来的） 可以和8种数据类型做运算 用   + ，表示相连接。  ⭐注意char用+做运算就是加（将char的字符转化为对应的码数再和其他数相连接），不是连接。
        接口（interface）
        数组（array）
二、 基本数据类型（7种数据类型，不包括Boolean）之间的运算规则
1、自动类型提升：
    结论：当内存容量小的数据类型和内存容量大的数据类型进行运算时，其结果自动转换为内存容量大的数据类型
    byte 、 char 、short -->int --> long -->float -->double
   ⭐ 特别的: 当byte 、 char 、 short 三种类型进行运算时, 结果最小也只能用int型接收
         例如    byte  a = 1； a = a+2；这是错误的
2、强制类型转换: 自动类型提升的逆过程
    格式:   新变量名 = (类型名int, float...)    变量名;
三、Java 种的进制
二进制(binary)：0,1 ，满2进1.以0b或0B开头。
十进制(decimal)：0-9 ，满10进1。
八进制(octal)：0-7 ，满8进1. 以数字0开头表示。 

十六进制(hex)：0-9及A-F，满16进1. 以0x或0X开头表示。此处的A-F不区分大小写。 如：0x21AF +1= 0X21B0
第三天
一、运算符

①
模运算，结果的正负主要和被取模数一样
②
++/–运算和其它运算一起出现时：
前++/–，先自生改变再进行其它运算
后++/–，先其它运算再改变自生的值
例如：int i = 10; i = i +(i++) + (++i) //i = 10+10+12这里的i++也是先赋值后加加
③
i += 1 这种运算是不改变i的类型 如果i是short型，则还是short型
和i = i+1 相比是有区别的，这里的i 如果是short型，则已经变成了int型了
④
逻辑运算符 &，&&，| , || , ! , ^ 逻辑与，短路与，逻辑或，短路或，非，异或
关于逻辑与和短路与的相同点：
相同点：两者的运算结果是相同的
不同点：逻辑与要考虑两边的情况，两边都运算完成。短路与是先考虑左边，如果左边已经能判定结果了，则右边就不运算了。
例如： boolean a = false；int b = 10;
if (a && (b++>0)) {} // 在这种情况下，a为false 则，&&结果一定是false 则右边就直接跳过了，使用b还是等于10
开发中推荐使用短路与/或
⑤位运算符
在一定范围内：左移相当于乘二，右移相当于除二
为什么要说一定范围呢，因为物极必反，比如向左移动时，当移到最高位为1时，值就变成负的了。

⑥三元运算符和if-else都可以用的时候用三元运算符好点
⑦比较运算符的输出结果为Boolean类型的，所以是不能连续使用的。例如：80< score <100是不行的。
比较运算符里面的 == ，！= 不仅可以用在数值类型数据之间，还可以运用在引用数据类型之间（如String），其它比如 > < 等只能用在数值类型数据之间。
第四天
一、if -else 结构的使用

①⭐当一个条件表达式成立，则进去执行相应的语句，后面即使又符合条件的语句也就不会再执行了 例如： if (score >90){执行语句1;}else (score>80){执行语句2;}，当score为91的时候，只会执行执行语句1,永远不会执行执行语句2。
所以得出以下几点：

如果多个条件表达式为互斥关系（没有交集），哪个判断语句和执行语句声明在上面和下面没关系
如果多个条件语句为交集的关系，需要根据实际情况，考虑清楚哪个结构在上，哪个在下（上面的例如中的情况）
如果多个条件语句为包换关系，通常情况下将范围小的声明在范围大的上面，这样范围小的才有机会执行。

二、在[a, b]区间内随机产生一个数
公式[a, b]：(int)(Math.random()*(b -a + 1)+a)
三、switch-case 的使用

①⭐根据switch表达式中的值，依次匹配各个case 中的常量。一旦匹配成功，则进入到相应的case结构中，调用其执行的语句。当调用完执行语句以后，则仍然向下执行其它case结构中的语句，直到遇见break关键字或者switch-casem末尾结束为止。
所以每个case结构中一般（是一般情况，有时候不加反而更方便，如下）加上break。

这样可以通过不写break，防止代码冗余。
所以说break在switch-case是可选的不是必须加的
②switch结构中的表达式， 只能有如下6中数据类型之一：byte ，short，char，int ，枚举类型，String类型。
③case之后只能是常量，不能是范围。例如：switch(age){case age>18:}这样是不行的，不然就和if-else很像了呀。所以switch-case一般应用在比较少的情况下。

四、if-else 和switch-case 使用情况说明
1、凡是可以switch-case都可以用if-else ，反之则不然
2、在写分支时，如果可以使用switc-case（并且switch中表达式的取值情况不多的时候），又可以使用if-else ，优先选择使用switch-case，原因：switch-case执行效率稍高

第五天
一、循环的四要素

①初始化条件，②循环条件（Boolean类型），③循环体，④迭代条件。
for（①；②；④）{③}
①while（②）{③；④}说明：别丢了迭代条件，避免死循环。
执行过程：① - ② - ③ - ④ - ② - ③ - ④ ………②
for循环和while循环可以相互转换
唯一的区别： for循环和while循环的初始化条件部分的作用范围不同，for循环的初始化只在for循环内有用。
do-while 循环结构
①do{③；④}while(②)
执行过程：① - ③ - ④ - ② - ③ - ④ …….②
do-while() 是一定会执行一次的循环体的！！！
在开发中使用for ，while() 结构多一些

二、一些小知识⭐⭐⭐

①不在循环部分限制次数的结构：for(;;)或while(true)
②结束循环的两种方式
方式一：循环条件部分返回false
方式二：在循环体中，执行break

三、break 默认跳出包裹此关键字最近的一层循环。
continue ，也是
 break可以用在循环和switch-case，continue 只能用在在循环中

⭐也可以跳出指定循环，只要再循环前面加上标签，然后再break或者continue后面加上标签，这样就结束/跳过指定标识的循环了。
