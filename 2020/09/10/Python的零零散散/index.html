Python的零零散散

发表于 2020-09-10

记录下在python中见到的一些不知道什么意思的东西。

①
def strStr(self, haystack: str, needle: str) -> int:
…
return a
函数标注通常用于 类型提示：以上函数预期接受两个str参数并预期返回一个 int 值。

②pop()：移除序列中的一个元素（默认最后一个元素），并且返回该元素的值。
移除list的元素，若元素序号超出list则会报错

.pop(),在()内为空时默认移除最后一个元素

list_1 = [1, 2, 3, 4, 5]

a = list_1.pop()

print (list_1, a)

–>[1, 2, 3, 4] 5

pop(元素序号)移除list中的某一个元素

list_1 = [1, 2, 3, 4, 5]

a = list_1.pop(2)

print (list_1, a)

–>[1, 2, 4, 5] 3

移除dict中的元素(只能移除1个元素)：pop()，()内不能为空，要有key；若key不在dict内，需要主动给出需要返回的数据，否则报错；

如果key在dict_1内

dict_1 = {1:’a’, 2:’b’, 3:’c’, 4:’d’, 5:’e’}

a = dict_1.pop(2)

print(dict_1, a)

–>{1:’a’, 3:’c’, 4:’d’, 5:’e’} b
如果key不在dict_1内：返回pop给定的数据（下面的[1,2,3,4]），一定要给定否则会报错

dict_1 = {1:’a’, 2:’b’, 3:’c’, 4:’d’, 5:’e’}

a = dict_1.pop(6, [1, 2, 3, 4])

print(dict_1, a)

–>{1:’a’, 2:’b’, 3:’c’, 4:’d’, 5:’e’} [1, 2, 3, 4]
